# 技术英文文档阅读 · 格式规范

## 单词查询格式

```
word
词性 · 中文意思

"英文例句"
"中文翻译"
```

**词性缩写**

| 缩写 | 词性 |
|------|------|
| n. | 名词 |
| v. | 动词 |
| adj. | 形容词 |
| adv. | 副词 |
| prep. | 介词 |

**近义词辨析**（仅在容易混淆时添加）

```
word
词性 · 中文意思

"英文例句"
"中文翻译"

vs. similar-word：区别说明
```

---

## 句子 / 段落翻译格式

```
English sentence or paragraph
中文翻译
```

段落按语义分组，每组之间空一行，英文在上，翻译在下，一次性返回。

---

## 使用规则

- 发单词 → 返回单词格式
- 发句子或段落 → 返回翻译格式
- 近义词混淆时 → 额外加 vs. 辨析
